DotArt

DotArt is a lightweight pixel art studio designed for Android devices. The goal of DotArt is to provide a fast and responsive pixel art creation environment that allows artists to focus on creativity instead of complex software interfaces. The application is optimized for low memory usage, fast startup times, and efficient rendering performance.

The architecture of DotArt is centered around modular drawing tools and JSON-based project storage, allowing artwork to be exported and shared easily between devices or collaborative workflows.

Technical Documentation

The Maneki software ecosystem follows a lightweight modular architecture that prioritizes simplicity, performance, and maintainability. Each application is built with minimal dependencies in order to maintain small binary sizes and fast startup performance.

Applications typically follow a layered architecture model that separates user interface components from core logic and data storage layers. This approach ensures that application features remain maintainable while allowing future extensions to be implemented with minimal changes to the existing codebase.

Typical internal structure used in Maneki applications:

UI Layer Handles rendering and user interaction. Logic Layer Processes commands and application state. Storage Layer Handles local data persistence and project files. Rendering Engine Used in creative applications such as DotArt for efficient pixel rendering.

Full documentation and developer notes are maintained in the project repository hosted on GitHub.
