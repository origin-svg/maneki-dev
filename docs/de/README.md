## Über das Unternehmen

Maneki & Co. LLC ist ein unabhängiges Software-Studio, das sich auf die Entwicklung leichter Anwendungen, experimenteller Tools und zugänglicher Web-Technologien spezialisiert hat.  
Unsere Philosophie legt Wert auf Einfachheit, Zuverlässigkeit, durchdachtes Design und Experimentierfreude in der Softwareentwicklung.  
Der Name **Maneki (招き)** stammt aus dem Japanischen und bedeutet *einladen* oder *willkommen heißen*, was das Ziel des Studios widerspiegelt, Kreativität und technologische Erforschung zu fördern.
