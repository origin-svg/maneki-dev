## À propos de l’entreprise

Maneki & Co. LLC est un studio de logiciels indépendant spécialisé dans le développement d’applications légères, d’outils expérimentaux et de technologies web accessibles.  
Notre philosophie met l’accent sur la simplicité, la fiabilité, le design réfléchi et l’expérimentation dans le développement logiciel.  
Le nom **Maneki (招き)** vient du japonais et signifie *inviter* ou *accueillir*, reflétant l’objectif du studio d’encourager la créativité et l’exploration technologique.

## Stack technologique

- Langages : JavaScript, Python, Java, Kotlin, C#, HTML, CSS, Lua

1. Fork du repo  
2. Créer une branche feature  
3. Suivre les conventions de code  
4. Soumettre un pull request
