## 🌐 English / Inglés

### [Viewify v1.0] 🚀
**Release Type:** Stable  
**Platform:** Android  
**Features Added:**
- Core app engine initialized  
- JSON-based project saving and loading  
- Multi-threaded rendering pipeline for fast sprite updates  
- Internalization added for repository and in-app menus (English + Spanish)  
- Basic user authentication and session management (local storage)  
- Versioning system integrated (SemVer 1.0.0)  

**Fixes:**
- Fixed crash on empty sprite load  
- Resolved memory leak in rendering engine  
- Corrected input mapping errors on touch devices  

**Experimental / Notes:**
- Placeholder for AI-assisted palette suggestions  
- Config files ready for multi-language expansions
