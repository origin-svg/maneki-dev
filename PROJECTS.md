## DotArt

**DotArt** is a pixel art studio designed for creating and editing pixel artwork on mobile devices.

Features include:

- Pixel canvas editor
- Color palette management
- JSON project format
- Import and export support
- Community shareable artwork files

Status: Public Beta  
Platform: Android

---

## Viewify

**Viewify** is a lightweight mobile application focused on simplicity and efficient performance.

The application is designed to provide a fast and minimal user experience.

Status: Active  
Platform: Android

---

## Purrfect Day

**Purrfect Day** is a relaxing cat-themed mobile application focused on simple and enjoyable interactions.

The application emphasizes a calm and playful experience.

Status: Development  
Platform: Android
