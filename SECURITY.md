# Security Policy

## Supported Versions

The following versions of Maneki & Co. LLC software currently receive security updates.

| Version | Supported |
|--------|-----------|
| Latest Release | Yes |
| Older Versions | No |

Only the most recent stable versions of our applications receive security updates.

---

## Reporting a Vulnerability

If you discover a security vulnerability in any software developed by **Maneki & Co. LLC**, please report it responsibly.

To report a vulnerability, please send an email with the following information:

- Description of the vulnerability
- Steps required to reproduce the issue
- Affected software and version
- Potential impact of the vulnerability
- Any proof-of-concept or screenshots (if applicable)

Security reports should be sent to:

**Email:** Next Level Com

---

## Responsible Disclosure

Please do not publicly disclose security vulnerabilities before they have been reviewed and addressed.

We appreciate responsible disclosure and will investigate all legitimate reports as quickly as possible.

---

## Scope

This security policy applies to software officially released by **Maneki & Co. LLC**, including but not limited to:

- Viewify
- DotArt
- Purrfect Day

Third-party dependencies or external software used by these projects may have separate security policies.

---

## Acknowledgements

We appreciate individuals who responsibly report security issues and help improve the safety of our software.
